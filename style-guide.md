
# Squabbl Style Guide

## 🎨 Color Palette

| Color Use            | Color Name         | Hex Code   | Notes                                      |
|---------------------|--------------------|------------|--------------------------------------------|
| Background          | Off-White          | #F8F8F8    | Clean, soft base for content               |
| Primary Accent      | Playful Coral      | #EF798A    | Used for buttons, highlights, key actions  |
| Secondary Accent    | Mint Green         | #B0EACD    | Complements coral, great for success states|
| Tertiary Accent     | Sunny Yellow       | #FFD166    | Use for highlights, alerts, fun moments    |
| Contrast/Depth      | Sky Blue           | #A8DADC    | Use for containers and section backgrounds |
| Text Primary        | Dark Slate Gray    | #2F4F4F    | High contrast but soft and inviting        |

## ✍️ Typography

| Use Case       | Font                   | Style/Weight       | Notes                                   |
|----------------|------------------------|---------------------|-----------------------------------------|
| Headings       | Poppins or Fredoka     | Bold / Semi-Bold    | Rounded, playful, modern                |
| Body Text      | Inter or Nunito        | Regular / Medium    | Clean, friendly, highly readable        |
| Labels/Buttons | Poppins or Nunito      | Medium / Bold       | Must be legible and stand out clearly   |

> **Tip:** Use uppercase only sparingly—titles or key buttons. Keep tone warm, not too “shouty.”

## 🖼️ Imagery & Iconography

- **Style:** Flat, playful 2D illustrations with slight shadows.
- **Shapes:** Rounded edges, soft curves, minimal angles.
- **Icons:** Friendly, outline style with filled highlights (match coral or mint).
- **Characters/People:** Expressive, diverse, abstracted cartoon forms for universal appeal.

## 🧩 UI Components

| Component      | Style Traits                                       |
|----------------|----------------------------------------------------|
| Buttons        | Rounded corners (2xl), shadow, coral/mint fill     |
| Cards          | Soft shadow, sky blue or mint container, padding   |
| Inputs         | Rounded fields, subtle borders, dark gray text     |
| Modal Dialogs  | Centered, lightly elevated with yellow highlights  |
| Timers/Score   | Coral highlights + dark slate text on light cards  |

## 📐 Layout & Spacing

- **Spacing:** 8pt grid (base spacing = 8px)
- **Containers:** Use max-width with centered content
- **Sections:** Clear breaks between steps/screens, big headings
- **Responsiveness:** Must work great on mobile—stacked layouts, buttons full-width

## 🎉 Tone & Personality

| Trait         | Description                                            |
|---------------|--------------------------------------------------------|
| Fun           | Bursts of color, animated moments, celebratory vibes  |
| Inclusive     | Easy-to-understand icons, warm language, welcoming UX |
| Competitive   | Scoreboards, team colors, game transitions feel alive |
| Surprising    | Cute animations when words guessed or rounds change   |
